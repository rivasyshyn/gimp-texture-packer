how to install: 
  copy script-fu-texture_packer.py into [gimp install dir]/lib/gimp/2.0/plug-ins/

how to launch:
  [menu]->Filters->TexturePacker
  
how it works:
  it takes all visible layers, autocrops each layer and pack them into new image

notes:
  doesn't handle layer groups
  algorithm of packing images very simple
